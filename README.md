## Usage

### bgmd

```bash
npm i bgmd
```

You can also just use the cdn `https://unpkg.com/bgmd@0/data/index.json`, `https://unpkg.com/bgmd@0/data/full.json`, or `https://unpkg.com/bgmd@0/data/calendar.json` to get the latest data.

### bgmc

```bash
npm i bgmc
```

### tmdbc

```bash
npm i tmdbc
```
